Ubuntu下用apt-get安装rails环境

Dec 17th, 2010

首先使用apt-get安装ruby

运行 ruby -v

ruby安装完毕

安装rubygems，注意,这里不要用apt-get来安装

先从官网下载最新版rubygems,并解压

运行安装程序

会看到输出很多文字，最后一行是

然后做一个ln连接
